# О проекте

Пользователь загружает файл с данными через веб-интерфейс. После этого агент получает доступ к структуре датасета и проводит анализ с помощью языковой модели DeepSeek V4 Pro через NVIDIA NIM API.

В процессе работы агент:

- изучает структуру набора данных;
- определяет типы признаков;
- выявляет пропущенные значения;
- анализирует статистические характеристики;
- формирует выводы о качестве данных;
- генерирует рекомендации для дальнейшего анализа.

---

# Основные возможности

### Загрузка данных

Поддерживаются форматы:

- CSV
- XLSX

### Исследование данных

Агент автоматически определяет:

- количество строк и столбцов;
- названия признаков;
- типы данных;
- наличие пропусков;
- статистические характеристики числовых признаков.

### Генерация отчёта

На основе результатов анализа формируется текстовый отчёт, содержащий:

- описание датасета;
- ключевые наблюдения;
- найденные проблемы;
- рекомендации по дальнейшей работе.

---

# Архитектура проекта

Проект реализован по агентной схеме.

Пользователь взаимодействует только с веб-интерфейсом, а дальнейший процесс анализа выполняется автоматически.

## Общая схема работы

```text
Пользователь
      │
      ▼
 Streamlit UI
      │
      ▼
 Загрузка CSV/XLSX
      │
      ▼
 Pandas DataFrame
      │
      ▼
 Data Agent
      │
      ▼
 DeepSeek V4 Pro
      │
      ▼
 Анализ данных
      │
      ▼
 Итоговый отчёт
```

---

# Используемая модель

Для анализа данных используется модель:

**DeepSeek V4 Pro**

Доступ к модели осуществляется через:

**NVIDIA NIM API**

---

# Установка

## Установка зависимостей

```bash
pip install -r requirements.txt
```

---

# Настройка API

Создайте файл `.env` в корне проекта:

```env
NVIDIA_API_KEY=your_api_key_here
```

Получить ключ можно в NVIDIA Build Platform.

---

# Запуск проекта

Запуск Streamlit-приложения:

```bash
python -m streamlit run app.py
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:8501
```
